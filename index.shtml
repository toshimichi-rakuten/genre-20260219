SSIを動作させるには

このサンプルを実際に動作させるには、SSIをサポートするウェブサーバーが必要です：

Apacheの場合:

  1. mod_includeモジュールを有効化
  2. .htaccessまたは設定ファイルに以下を追加: 
    Options +Includes
AddType text/html .shtml
AddOutputFilter INCLUDES .shtml

開発環境でテストする場合:

npm パッケージ ssi-server を使用できます:

npm install -g ssi-server
ssi-server -p 3000
